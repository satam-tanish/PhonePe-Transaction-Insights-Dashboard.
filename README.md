# 📊 PhonePe Transaction Insights Dashboard

## 🚀 Project Overview
The **PhonePe Transaction Insights Dashboard** is a data analytics project that explores digital payment trends across India using the PhonePe Pulse dataset.

This project extracts raw JSON data, processes it using Python, and presents insights through an interactive Streamlit dashboard.

---

## 🎯 Objectives
- Analyze transaction trends across different states in India
- Identify top-performing states and payment categories
- Visualize digital payment growth
- Build an interactive dashboard for data exploration

---

## 🛠️ Tech Stack
- **Python**
- **Pandas**
- **Streamlit**
- **Matplotlib**
- **Seaborn**
